# Global Health ETL Pipeline

## Overview
This project is an ETL (Extract, Transform, Load) pipeline designed to process global health data. 

## Project Architecture
1. **Extract**: Data is extracted from Hugging Face datasets.
2. **Transform**: The extracted data is cleaned and transformed using Python and Pandas.
3. **Load**: The transformed data is loaded into a PostgreSQL database.
4. **Visualization (Upcoming)**: Later stages of the project will involve creating visualizations to derive insights from the processed data.

## Tech Stack
- **Python**
- **Pandas** (Data Transformation)
- **Hugging Face Datasets** (Data Source)
- **PostgreSQL** (Data Storage)
